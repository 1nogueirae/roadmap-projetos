# Restricted Textarea

A simple dynamic textarea with real-time character counter.

## Features

- **Auto-expanding textarea** - grows as you type
- **Character counter** - displays current character count inside the textarea
- **Responsive design** - works on different screen sizes

## How it works

The textarea automatically adjusts its height based on the content and shows a character counter in the bottom-right corner.

Based on the https://roadmap.sh/projects/restricted-textarea project.

## Files

- `script.js` - JavaScript logic for textarea behavior and character counting
- `style.css` - Styling and layout
- `index.html` - HTML structure

## Usage

1. Open `index.html` in your browser
2. Start typing in the textarea
3. The height will adjust automatically and the character count will update in real-time
