# Simple Tabs

A simple tab system created with HTML, CSS and vanilla JavaScript.

## Description

This is a basic project that demonstrates how to create a tab navigation system using only HTML, CSS and vanilla JavaScript, with no external dependencies.

The project is based on the roadmap.sh tutorial: https://roadmap.sh/projects/simple-tabs

## Features

- 4 navigable tabs
- Clean and responsive interface
- Pure JavaScript, no libraries

## How to Use

1. Open the `index.html` file in your browser
2. Click the buttons to switch between different tabs

## Files

- `index.html` - HTML page structure
- `style.css` - Styles and layout
- `script.js` - Tab switching logic

## Technologies

- HTML5
- CSS3
- JavaScript
