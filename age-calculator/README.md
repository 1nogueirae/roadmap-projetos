# Age Calculator

A simple web application that calculates your age based on your birth date.

## Features

- Select your birth date using an interactive date picker
- Calculate your exact age in years, months, and days
- Clean and simple user interface
- Input validation for date selection

## How to Use

1. Click on the input field and select your birth date
2. Click the "Calculate Age" button
3. Your age will be displayed in years, months, and days

## Technologies Used

- **HTML5** - Page structure
- **CSS** - Styling
- **JavaScript** - Logic and calculations
- **Luxon** - Date and time library for precise age calculations
- **js-datepicker** - Interactive date picker component
