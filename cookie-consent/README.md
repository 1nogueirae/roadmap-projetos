# Cookie Consent

Simple cookie consent popup implementation.

## Description

A basic cookie consent banner that appears at the bottom-right corner of the page. Users can accept or close the popup.

The project is based on Roadmap to learn JavaScript, following the tutorial on this link: https://roadmap.sh/projects/cookie-consent

## How to Use

Open `index.html` in your browser.

## Technologies

- HTML
- CSS
- JavaScript
